list of 5 HTML tags with thier description

thead
The table header(thead) is used to add a heading row for the table columns. This is where we add labels for the data being displayed. Using the thead opening and closing tags
ul
We use ul tag for the unordered list. In detail, it is used when the items inside the list do not have a specific order or the order is not important.
ol
We use ol tag when we want to put ordered lists , whenever the order of items in the list is important.
table
We use the table row tag(tr) to add a table to the page
a
The a tag is called anchor tag. We use the anchor tag for creating a link on our webpage
